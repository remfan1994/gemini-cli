NAME
    gemini-cli - terminal-based Gemini chat interface

SYNOPSIS
    gemini-cli
    gemini-cli <session_name>
    gemini-cli --resume <session_name>
    gemini-cli --list
    gemini-cli --version
    gemini-cli --help
    gemini-cli -h

DESCRIPTION
    gemini-cli is a lightweight terminal chat interface
    for the Google Gemini API.

    The tool is designed for:
        - minimal dependencies
        - persistent local sessions
        - readable terminal interaction
        - artifact-oriented workflows

FEATURES
    - Persistent chat sessions
    - Named/resumable conversations
    - Timestamped logs
    - Configurable model selection
    - Lightweight rolling memory
    - Automatic code attachment extraction
    - Configurable storage directories

HELP SYSTEM
    Built-in usage help is available directly from the
    command line via:

        gemini-cli --help
        gemini-cli -h

SESSION MANAGEMENT
    gemini-cli
        Start new timestamped session.

    gemini-cli <name>
        Start/create named session.

    gemini-cli --resume <name>
        Resume named session.

    gemini-cli --list
        List saved sessions.

    gemini-cli --version
        Display version information.

    gemini-cli --help
    gemini-cli -h
        Display built-in help summary.

CONFIGURATION
    Config file:
        ~/.config/gemini-cli/config

    Format:
        KEY=VALUE

    Supported keys:

        MODEL
            Gemini model identifier.

        HISTORY_LIMIT
            Rolling context window size.

        GEMINI_API_KEY
            Optional API key.

        SESSION_DIR
            Directory used for session log storage.

        ATTACHMENT_DIR
            Directory used for extracted code artifacts.

ENVIRONMENT
    GEMINI_API_KEY
        Preferred API key source.

        Environment variables override config values.

ATTACHMENT SYSTEM
    Code blocks returned by Gemini are automatically
    extracted into files under ATTACHMENT_DIR.

    Supported attachment types include:

        bash/sh      -> .sh
        python       -> .py
        json         -> .json
        yaml/yml     -> .yml
        javascript   -> .js
        html         -> .html
        css          -> .css
        sql          -> .sql
        unknown      -> .txt

    Terminal output replaces code blocks with:

        [attachment saved]

FILES
    SESSION_DIR
        Session logs

    ATTACHMENT_DIR
        Extracted code artifacts

    ~/.config/gemini-cli/config
        User configuration file

MEMORY MODEL
    Maintains lightweight rolling conversation context.

LIMITATIONS
    - Requires internet access
    - No readline integration
    - API quota restrictions apply
