NAME
    gemini-cli - terminal-based Gemini chat interface

SYNOPSIS
    gemini-cli
    gemini-cli <session_name>
    gemini-cli --resume <session_name>
    gemini-cli --list
    gemini-cli --version

DESCRIPTION
    gemini-cli is a lightweight terminal chat client for the Google
    Gemini API. It provides persistent sessions, timestamped logs,
    configurable settings, and lightweight in-session memory.

SESSION MANAGEMENT
    gemini-cli
        Start a new timestamped session.

    gemini-cli <name>
        Start or create a named session stored as:
            ~/.gpt/<name>.log

    gemini-cli --resume <name>
        Resume an existing session.

    gemini-cli --list
        List saved sessions.

    gemini-cli --version
        Display version information.

CONFIGURATION
    Config file location:
        ~/.config/gemini-cli/config

    Supported keys:

        MODEL
            Model name used for generation.
            Default: gemini-2.5-flash

        HISTORY_LIMIT
            Number of recent messages kept in memory.

        GEMINI_API_KEY
            Optional API key (env overrides this value).

ENVIRONMENT
    GEMINI_API_KEY
        Recommended method for supplying API key.

        Precedence:
            1. Environment variable (highest priority)
            2. Config file

VISUAL STRUCTURE
    Conversations are visually separated:

        END INPUT
        BEGIN OUTPUT

    Each message includes timestamps.

MEMORY MODEL
    - Short-term sliding history window
    - Lightweight rolling summary of older context

FILES
    ~/.gpt/
        Session logs

    ~/.config/gemini-cli/config
        User configuration file

LIMITATIONS
    - Requires internet access
    - No readline/rlwrap support (basic terminal input only)
    - API quota limits apply
