NAME
    gemini-cli - terminal-based Gemini chat interface

SYNOPSIS
    gemini-cli
    gemini-cli <session_name>
    gemini-cli --resume <session_name>
    gemini-cli --list
    gemini-cli --version

DESCRIPTION
    gemini-cli is a lightweight terminal chat client for the Google
    Gemini API. It provides persistent sessions, timestamped logs,
    configurable settings, and lightweight in-session memory.

SESSION MANAGEMENT
    gemini-cli
        Start a new timestamped session.

    gemini-cli <name>
        Start or create a named session stored as:
            ~/.gpt/<name>.log

    gemini-cli --resume <name>
        Resume an existing named session.

    gemini-cli --list
        List available saved sessions.

    gemini-cli --version
        Display installed version.

CONFIGURATION
    Configuration file:
        ~/.config/gemini-cli/config

    Supported keys:

        MODEL
            Gemini model name (default: gemini-2.5-flash)

        HISTORY_LIMIT
            Number of recent messages kept in memory

VISUAL STRUCTURE
    Messages are separated using clear blocks:

        END INPUT
        BEGIN OUTPUT

    Each message is timestamped for clarity.

MEMORY MODEL
    The system maintains:
        - short-term message history window
        - lightweight rolling summary of older context

FILES
    ~/.gpt/
        Session logs

    ~/.config/gemini-cli/config
        User configuration file

ENVIRONMENT
    GEMINI_API_KEY
        Required API key for Gemini access

LIMITATIONS
    - Requires internet access
    - No readline/rlwrap dependency (limited line editing)
    - API quota restrictions apply
